## 🍲 新疆风味牛肉炖土豆（无豆瓣酱、无西红柿版）

### 🧂 食材准备（约3-4人份）

| 食材      | 用量                |
| ------- | ----------------- |
| 牛腱子或牛肩肉 | 500 克             |
| 土豆      | 3 个中等大小（土豆建议用粉糯型） |
| 洋葱      | 1 个（中等）           |
| 干辣椒     | 5-8 个（可按口味调整）     |
| 花椒      | 1 小把（约20粒）        |
| 大蒜      | 6-8 瓣（拍碎）         |
| 生姜      | 1 小块（拍碎）          |
| 八角      | 2 个               |
| 小茴香     | 1 茶匙（可略多些）        |
| 肉桂      | 1 小段（可选）          |
| 老抽      | 1 茶匙（上色）          |
| 生抽      | 2 汤匙（调味）          |
| 盐       | 适量                |
| 白胡椒粉    | 适量                |
| 清水      | 适量（约800ml）        |
| 食用油     | 适量                |
| 香菜      | 少许（可选装饰）          |

---

### 🔪 前期处理

1. **牛肉切块**：切 3\~4cm 的大块，冷水下锅，焯水去血沫。捞出冲净备用。
2. **土豆切块**：切滚刀块，泡水防氧化。
3. **洋葱切块**：切大块或丝，备用。
4. **香料准备**：花椒、八角、小茴香、肉桂提前用清水泡一下去除灰尘。
5. **干辣椒剪段**：不喜太辣可以去籽。

---

### 🍳 正式烹饪步骤

#### 1. 爆香香料

* 热锅凉油，放入花椒、八角、小茴香、干辣椒、肉桂，小火慢慢炒香（别糊了）。
* 下入拍碎的大蒜、姜片，炒出香味。
* 放入洋葱炒至微透明。

#### 2. 煸炒牛肉

* 加入焯水后的牛肉块，大火煸炒至牛肉微微变色、边缘微焦。
* 加入 **老抽、生抽**，翻炒均匀使牛肉上色。

#### 3. 加水炖煮

* 加入清水没过食材 2cm 左右，水开后转小火炖 **60-90 分钟**（视牛肉老嫩）。
* 盖锅焖炖至牛肉酥烂（筷子能轻松插入）。

#### 4. 加入土豆

* 加入土豆块，继续炖 20 分钟左右，直到土豆熟透、软糯但不碎。
* 最后调入盐和适量白胡椒粉。

#### 5. 收汁

* 汤汁略多可开中火收汁，至粘稠但不干锅状态为佳。

#### 6. 装盘撒香菜（可选）

---

### 🍽️ 小贴士

* 不喜欢太油可以在收汁前撇掉多余浮油。
* 可搭配馕、米饭或拉条子食用。
* 若喜欢更新疆一点的味道，可以加一点孜然粉（但不加也很地道，靠香料本身就足够）。
