## 📖 内容管理

### 添加新内容

1. **创建新文档**：在相应目录下创建 `.md` 文件
2. **使用 Markdown**：所有文档都使用 Markdown 格式
3. **添加标签**：使用 `type:` 标签进行分类
4. **保持结构**：遵循现有的文档结构

### 文档格式

每个食谱文档建议包含：

```markdown
# 菜名

type: 分类标签

## 食材准备

## 制作步骤

## 小贴士
```
